**2025 年 9 月** 是 Supermemory 的重要月份：

<div id="for-developers">
  ### 面向开发者
</div>

* **文档全面重构** - 全新文档，包含实用“cookbook”范例与生产级示例
* **原生 AI SDK 集成** - 可直接与 Vercel AI SDK 搭配使用 Supermemory
* **强化 API** - 新增批量操作与优化的搜索端点

<div id="for-users">
  ### 面向用户
</div>

* **可视化控制台全面升级** - 全新瀑布流布局与焕新的预览卡片
* **更佳的 memory 浏览体验** - 交互式图谱可视化，具备 WebGL 级性能
* **移动端体验优化** - 所有功能均具备更佳的响应式表现
