构建一个个人 AI 助手，能够学习并记住有关用户的一切——他们的偏好、习惯、工作背景与对话历史。此示例演示如何使用 Supermemory 的 Memory Tools 打造真正个性化的 AI 体验。

<div id="what-youll-build">
  ## 你将构建的内容
</div>

一个个人 AI 助手，它可以：

- **记住用户偏好**（饮食禁忌、工作安排、沟通风格）
- **从对话中不断学习**，并随着时间改进回复
- **在多次聊天会话间保持上下文**
- **基于用户历史提供个性化推荐**
- **在保持上下文的同时处理多个话题**

    ```python main.py
    from fastapi import FastAPI, HTTPException
    from fastapi.responses import StreamingResponse
    import openai
    from supermemory import Supermemory
    import json
    import os
    from typing import List, Dict, Any
    import asyncio

    app = FastAPI()

    openai_client = openai.AsyncOpenAI(api_key=os.getenv("OPENAI_API_KEY"))
    supermemory_client = Supermemory(api_key=os.getenv("SUPERMEMORY_API_KEY"))

    SYSTEM_PROMPT = """你是一名高度个性化的 AI 助手。你的首要目标是了解用户，并随着时间的推移提供愈加个性化的帮助。

    记忆管理：
    1. 当用户分享个人信息、偏好或上下文时，立即存储
    2. 在回应请求之前，先搜索与该用户相关的上下文
    3. 使用过往对话来指导当前回复
    4. 记住用户的沟通风格、偏好和常聊话题

    人格与风格：
    - 调整你的沟通方式以贴合用户偏好
    - 在合适的情况下自然引用过往对话
    - 基于已学习的模式主动提供帮助
    - 在尊重隐私的前提下真诚且有用

    在回复之前务必先搜索记忆，以提供个性化、具备上下文的帮助。"""

    async def search_user_memories(query: str, user_id: str) -> str:
        """在用户的记忆中搜索相关上下文"""
        try:
            results = supermemory_client.search.memories(
                q=query,
                container_tag=f"user_{user_id}",
                limit=5
            )

            if results.results:
                context = "\n".join([r.memory for r in results.results])
                return f"关于该用户的相关记忆：\n{context}"
            return "未找到相关记忆。"
        except Exception as e:
            return f"搜索记忆时出错：{e}"

    async def add_user_memory(content: str, user_id: str):
        """向用户的记忆添加新信息"""
        try:
            supermemory_client.memories.add(
                content=content,
                container_tag=f"user_{user_id}",
                metadata={"type": "personal_info", "timestamp": "auto"}
            )
        except Exception as e:
            print(f"添加记忆时出错：{e}")

    @app.post("/chat")
    async def chat_endpoint(data: dict):
        messages = data.get("messages", [])
        user_id = data.get("userId", "default-user")

        if not messages:
            raise HTTPException(status_code=400, detail="No messages provided")
            raise HTTPException(status_code=400, detail="未提供消息")
        # 获取用户的最后一条消息以进行记忆搜索
        user_message = messages[-1]["content"] if messages else ""

        # 搜索相关记忆
        memory_context = await search_user_memories(user_message, user_id)

        # 使用记忆上下文添加系统消息
        enhanced_messages = [
            {"role": "system", "content": f"{SYSTEM_PROMPT}\n\n{memory_context}"}
        ] + messages

        try:
            response = await openai_client.chat.completions.create(
                model="gpt-5",
                messages=enhanced_messages,
                stream=True,
                temperature=0.7
            )

            async def generate():
                full_response = ""
                async for chunk in response:
                    if chunk.choices[0].delta.content:
                        content = chunk.choices[0].delta.content
                        full_response += content
                        yield f"data: {json.dumps({'content': content})}\n\n"

                # 在回复完成后，分析是否包含值得存入记忆的内容
                if "remember" in user_message.lower() or any(word in user_message.lower() for word in ["prefer", "like", "dislike", "work", "schedule", "diet"]):
                    await add_user_memory(user_message, user_id)

            return StreamingResponse(generate(), media_type="text/plain")

        except Exception as e:
            raise HTTPException(status_code=500, detail=str(e))

    if __name__ == "__main__":
        import uvicorn
        uvicorn.run(app, host="0.0.0.0", port=8000)
    ```

<div id="troubleshooting">
  ## 疑难解答
</div>

**Memory 未被持久化？**

- 检查是否始终使用一致的 `x-sm-user-id` 请求头
- 验证 API 密钥是否具备写入权限
- 确保容器标签配置正确

**响应缺乏个性化？**

- 增大搜索 limit 以检索更多相关的记忆存储
- 降低 最小相似度阈值 扩大匹配范围
- 检查是否在正确的上下文下添加了记忆存储

**性能问题？**

- 减小搜索 limit 以提升响应速度
- 为高频搜索实现缓存
- 使用合适的 最小相似度阈值 在速度与准确性之间取得平衡
