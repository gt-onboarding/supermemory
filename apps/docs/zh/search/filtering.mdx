Supermemory 支持在 `/v3/search` 和 `/v4/search` 端点使用容器标签、Metadata 条件以及高级过滤技术来筛选搜索结果。

<div id="container-tags">
  ## 容器标签
</div>

容器标签可按用户、项目或组织对记忆存储进行分组。它们是隔离搜索结果的主要方式。

**重要**：容器标签采用**精确数组匹配**。具有 `["technology", "quantum-computing"]` 的文档不会匹配对 `["technology"]` 的搜索。数组必须完全一致。

<div id="document-search-v3search">
  ### Documents 搜索（v3/search）
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // 单个 container tag
    const results = await client.search.documents({
      q: "machine learning",
      containerTags: ["user_123"],
      limit: 10
    });

    // 多个 container tags
    const results2 = await client.search.documents({
      q: "project status",
      containerTags: ["project_ai", "team_research"],
      limit: 10
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # 单个 container tag
    results = client.search.documents(
        q="machine learning",
        container_tags=["user_123"],
        limit=10
    )

    # 多个 container tags
    results2 = client.search.documents(
        q="project status",
        container_tags=["project_ai", "team_research"],
        limit=10
    )
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # 单个 container tag
    curl -X POST "https://api.supermemory.ai/v3/search" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "q": "machine learning",
        "containerTags": ["user_123"],
        "limit": 10
      }'
    ```
  </Tab>
</Tabs>

<Note>
**Container 标签差异**：
- `/v3/search` 使用 `containerTags`（复数数组），采用精确数组匹配
- `/v4/search` 使用 `containerTag`（单个字符串），用于单标签过滤
- 精确匹配意味着 `["user", "project"]` ≠ `["user"]`
</Note>

<div id="filter-best-practices">
  ## 过滤最佳实践
</div>

<Callout type="info">
**性能建议**：
- 优先使用容器标签作为主过滤条件（速度最快）
- 搭配简单的 Metadata 过滤提升精度
- 为提升性能，避免使用复杂的嵌套 OR 条件
- 仅在必要时使用数值比较运算符
</Callout>

<Warning>
**过滤结构**：所有条件都必须包裹在 `AND` 或 `OR` 数组中。即使只有一个条件也需要数组包装：`{"AND": [{"key": "category", "value": "ai"}]}`。
</Warning>

<Note>
**支持的运算符**：
- **字符串**：仅支持精确匹配
- **数值**：`<=`、`>=`、`<`、`>`、`=`
- **数组**：使用 `array_contains` 检查数组是否包含指定值
- **取反**：适用于所有过滤类型
</Note>
