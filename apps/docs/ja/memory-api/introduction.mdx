**supermemory** は**データの取り込みと処理**を自動化し、検索可能にします。

<Info>
  Memory エンジンは線形にスケールします — つまり、**非常に高速でスケーラブル**でありながら、最も手頃な選択肢の一つを提供します
</Info>

![supermemory](/images/processing.png)

また、次のような機能も提供します:

* [コネクタと同期](/ja/memory-api/connectors/)
* [マルチモーダル対応](/ja/memory-api/features/auto-multi-modal)
* [高度なフィルタリング](/ja/memory-api/features/filtering)
* [Reranking](/ja/memory-api/features/reranking)
* [テキストからの詳細抽出](/ja/memory-api/features/content-cleaner)
* [クエリ リライト](/ja/memory-api/features/query-rewriting)

... そのほかにも多数あります!
