ファイルを Supermemory に直接アップロードして、コンテンツを自動で抽出・処理します。

<div id="upload-a-pdf">
  ## PDF をアップロード
</div>

OCR に対応して PDF からテキストを抽出します。

<CodeGroup>

```typescript TypeScript
const file = fs.createReadStream('document.pdf');

const response = await client.memories.uploadFile({
  file: file,
  containerTags: 'documents'
});

console.log(response.id);
// 出力例: pdf_123
```

```python Python
with open('document.pdf', 'rb') as file:
    response = client.memories.upload_file(
        file=file,
        container_tags='documents'
    )

print(response['id'])
# 出力例: pdf_123
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/documents/file" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -F "file=@document.pdf" \
  -F "containerTags=documents"

# レスポンス: {"id": "pdf_123", "status": "processing"}
```

</CodeGroup>

<div id="browser-file-upload">
  ## ブラウザでのファイルアップロード
</div>

ブラウザからのファイルアップロードを処理します。

<CodeGroup>

```javascript JavaScript
const formData = new FormData();
formData.append('file', fileInput.files[0]);
formData.append('containerTags', 'uploads');

const response = await fetch('https://api.supermemory.ai/v3/documents/file', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${API_KEY}`
  },
  body: formData
});

const result = await response.json();
console.log(result.id);
```

```typescript React
function handleUpload(file: File) {
  const formData = new FormData();
  formData.append('file', file);
  formData.append('containerTags', 'uploads');

  return fetch('https://api.supermemory.ai/v3/documents/file', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${API_KEY}` },
    body: formData
  });
}
```

```bash cURL
# ブラウザからのアップロードは FormData を使用します（ファイルアップロードと同様）
curl -X POST "https://api.supermemory.ai/v3/documents/file" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -F "file=@document.pdf" \
  -F "containerTags=uploads"
```

</CodeGroup>

<div id="upload-multiple-files">
  ## 複数ファイルをアップロードする
</div>

レート制限付きのバッチアップロード。

<CodeGroup>

```typescript TypeScript
for (const file of files) {
  const stream = fs.createReadStream(file);

  await client.memories.uploadFile({
    file: stream,
    containerTags: 'batch'
  });

  // レート制限
  await new Promise(r => setTimeout(r, 1000));
}
```

```python Python
import time

for file_path in files:
    with open(file_path, 'rb') as file:
        client.memories.upload_file(
            file=file,
            container_tags='batch'
        )

    time.sleep(1)  # レート制限
```

```bash cURL
# 各ファイルを間隔を空けて個別にアップロード
for file in *.pdf; do
  curl -X POST "https://api.supermemory.ai/v3/documents/file" \
    -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
    -F "file=@$file" \
    -F "containerTags=batch"

  sleep 1  # レート制限
done
```

</CodeGroup>

<div id="supported-file-types">
  ## サポート対象のファイル形式
</div>

<div id="documents">
  ### documents
</div>

| フォーマット | 拡張子 | 処理 |
|--------|------------|------------|
| PDF | .pdf | テキスト抽出、スキャンページの OCR |
| Microsoft Word | .doc, .docx | 全文および書式の抽出 |
| プレーンテキスト | .txt, .md | 直接テキストを処理 |
| CSV | .csv | 構造化データの抽出 |

<div id="images">
  ### 画像
</div>

| 形式 | 拡張子 | 処理 |
|--------|------------|------------|
| JPEG | .jpg, .jpeg | OCR（光学文字認識）でテキスト抽出 |
| PNG | .png | OCR（光学文字認識）でテキスト抽出 |
| GIF | .gif | 静止画像に対するOCR（光学文字認識） |
| WebP | .webp | OCR（光学文字認識）でテキスト抽出 |

<div id="size-limits">
  ### サイズ制限
</div>

- **最大ファイルサイズ**: 50MB
- **推奨サイズ**: 最適な処理のために10MB未満
- **大容量ファイル**: 処理に時間がかかる場合があります
