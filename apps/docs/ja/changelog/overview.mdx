<div id="whats-new-this-month">
  ## 今月のアップデート
</div>

**2025年9月**は、Supermemory にとって大きな節目の月となりました。

<div id="for-developers">
  ### 開発者向け
</div>

* **ドキュメントを全面改訂** - クックブック形式のレシピや本番運用の実例を備えた新しいドキュメント
* **ネイティブな AI SDK 統合** - Vercel AI SDK から Supermemory を直接利用
* **API の強化** - 新たな一括処理や最適化された検索エンドポイント

<div id="for-users">
  ### ユーザー向け
</div>

* **ビジュアルコンソールの刷新** - 新しいメイソンリー配置と再設計されたプレビューCard
* **メモリーの閲覧性向上** - WebGL級のパフォーマンスによるインタラクティブなグラフ可視化
* **モバイルでの使い勝手向上** - 全機能でのレスポンシブデザインを改善
