<div id="whats-new-this-month">
  ## Nouveautés de ce mois-ci
</div>

**Septembre 2025** a été un mois important pour Supermemory :

<div id="for-developers">
  ### Pour les développeurs
</div>

* **Réécriture complète de la documentation** - Nouvelle doc avec des recettes pratiques et des exemples prêts pour la production
* **Intégration native du SDK d’IA** - Utilisez Supermemory directement avec Vercel AI SDK
* **API améliorées** - Nouvelles opérations par lots et points de terminaison de recherche optimisés

<div id="for-users">
  ### Pour les utilisateurs
</div>

* **Refonte de la console visuelle** - Nouveau layout en maçonnerie et cartes de prévisualisation repensées
* **Navigation dans les memory améliorée** - Visualisation de graphes interactive avec des performances WebGL
* **Expérience mobile améliorée** - Meilleure conception responsive sur l’ensemble des fonctionnalités
