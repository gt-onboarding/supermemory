<div id="usage">
  ## Utilisation
</div>

```python
import os
from supermemory import Supermemory

client = supermemory(
    api_key=os.environ.get("SUPERMEMORY_API_KEY"),  # Ceci est la valeur par défaut et peut être omise
)

response = client.search.documents(
    q="documents liés à python",
)
print(response.results)
```

<div id="usage">
  ## Utilisation
</div>

```js
import supermemory from 'supermemory';

const client = new supermemory({
  apiKey: process.env['SUPERMEMORY_API_KEY'], // Ceci est la valeur par défaut et peut être omise
});

async function main() {
  const response = await client.search.documents({ q: 'documents liés à python' });

  console.debug(response.results);
}

main();
```
