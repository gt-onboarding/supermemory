---
title: "Statut de traitement"
description: "Découvrez les étapes du traitement du contenu"
---

Après avoir ajouté du contenu, vous pouvez vérifier son status de traitement :

1. `queued`: Le contenu est en attente de traitement
2. `extracting`: Extraction du contenu depuis la source
3. `chunking`: Division du contenu en segments sémantiques
4. `embedding`: Génération d’embeddings vectoriels
5. `indexing`: Ajout à l’index de recherche
6. `done`: Traitement terminé
7. `failed`: Échec du traitement
