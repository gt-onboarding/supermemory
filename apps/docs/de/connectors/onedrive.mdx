Verbinde OneDrive, um Word-Dokumente, Excel-Arbeitsmappen und PowerPoint-Präsentationen automatisch mit deiner Supermemory-Wissensbasis zu synchronisieren. Unterstützt sowohl persönliche als auch Geschäftskonten mit zeitgesteuerter Synchronisierung.

<div id="quick-setup">
  ## Schnellstart
</div>

<div id="1-create-onedrive-connection">
  ### 1. OneDrive‑Verknüpfung erstellen
</div>

<CodeGroup>

```typescript Typescript
import Supermemory from 'supermemory';

const client = new Supermemory({
  apiKey: process.env.SUPERMEMORY_API_KEY!
});

const connection = await client.connections.create('onedrive', {
  redirectUrl: 'https://yourapp.com/auth/onedrive/callback',
  containerTags: ['user-123', 'onedrive-sync'],
  documentLimit: 1500,
  metadata: {
    source: 'onedrive',
    accountType: 'business',
    department: 'marketing'
  }
});

// Nutzer zu Microsoft OAuth weiterleiten
window.location.href = connection.authLink;
// Ausgabe: Leitet zum OAuth-Provider weiter
// Ausgabe: Leitet zu https://login.microsoftonline.com/oauth2/authorize?... weiter
```

```python Python
from supermemory import Supermemory
import os

client = Supermemory(api_key=os.environ.get("SUPERMEMORY_API_KEY"))

connection = client.connections.create(
    'onedrive',
    redirect_url='https://yourapp.com/auth/onedrive/callback',
    container_tags=['user-123', 'onedrive-sync'],
    document_limit=1500,
    metadata={
        'source': 'onedrive',
        'accountType': 'business',
        'department': 'marketing'
    }
)

# Nutzer zu Microsoft OAuth weiterleiten
print(f'Redirect to: {connection.auth_link}')
# Ausgabe: Redirect to: https://oauth.provider.com/...
# Ausgabe: Redirect to: https://login.microsoftonline.com/oauth2/authorize?...
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/connections/onedrive" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "redirectUrl": "https://yourapp.com/auth/onedrive/callback",
    "containerTags": ["user-123", "onedrive-sync"],
    "documentLimit": 1500,
    "metadata": {
      "source": "onedrive",
      "accountType": "business",
      "department": "marketing"
    }
  }'

# Antwort: {
#   "authLink": "https://login.microsoftonline.com/oauth2/authorize?...",
#   "expiresIn": "1 hour",
#   "id": "conn_od123"
# }
```

</CodeGroup>

<div id="sync-mechanism">
  ## Synchronisierungsmechanismus
</div>

Webhooks sorgen für die sofortige Synchronisierung von Änderungen in documents. Sie können die Synchronisierung auch manuell auslösen.

<div id="manual-sync-trigger">
  ### Manuelles Synchronisieren auslösen
</div>

<CodeGroup>
  ```typescript Typescript
    // Sofortige Synchronisierung für alle OneDrive-Verknüpfungen auslösen
    await client.connections.import('onedrive');

    // Synchronisierung für einen bestimmten Benutzer auslösen
    await client.connections.import('onedrive', {
      containerTags: ['user-123']
    });

    console.log('Manuelle Synchronisierung gestartet – documents werden innerhalb von 5–10 Minuten aktualisiert');
    ```
  ```python Python
    # Sofortige Synchronisierung für alle OneDrive-Verknüpfungen auslösen
    client.connections.import_('onedrive')

    # Synchronisierung für einen bestimmten Benutzer auslösen
    client.connections.import_(
        'onedrive',
        container_tags=['user-123']
    )

    print('Manuelle Synchronisierung gestartet – documents werden innerhalb von 5–10 Minuten aktualisiert')
    ```
  ```bash cURL
    # Manuelle Synchronisierung auslösen
    curl -X POST "https://api.supermemory.ai/v3/connections/onedrive/import" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"containerTags": ["user-123"]}'
    ```
</CodeGroup>

<div id="delete-connection">
  ## Verbindung löschen
</div>

Entfernen Sie ein OneDrive‑verknüpftes Konto, wenn es nicht mehr benötigt wird:

<CodeGroup>

```typescript Typescript
// Nach Verbindungs-ID löschen
const result = await client.connections.delete('connection_id_123');
console.log('Verbindung gelöscht:', result.id);

// Nach Provider und Container-Tags löschen
const providerResult = await client.connections.deleteByProvider('onedrive', {
  containerTags: ['user-123']
});
console.log('OneDrive-Verbindung für Nutzer gelöscht');
```

```python Python
# Nach Verbindungs-ID löschen
result = client.connections.delete('connection_id_123')
print(f'Verbindung gelöscht: {result.id}')

# Nach Provider und Container-Tags löschen
provider_result = client.connections.delete_by_provider(
    'onedrive',
    container_tags=['user-123']
)
print('OneDrive-Verbindung für Nutzer gelöscht')
```

```bash cURL
# Nach Verbindungs-ID löschen
curl -X DELETE "https://api.supermemory.ai/v3/connections/connection_id_123" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY"

# Nach Provider und Container-Tags löschen
curl -X DELETE "https://api.supermemory.ai/v3/connections/onedrive" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"containerTags": ["user-123"]}'
```

</CodeGroup>

<Note>
Das Löschen eines verknüpften Kontos:
- Stoppt alle zukünftigen Synchronisierungen von OneDrive
- Entfernt die OAuth-Autorisierung
- Behält bereits synchronisierte documents in Supermemory (sie werden nicht gelöscht)
</Note>

  ```typescript Typescript
      metadata: { customApp: true }
    ```


<div id="optimization-tips">
  ### Optimierungstipps
</div>

1. **Realistische Dokumentlimits festlegen** basierend auf Speicher und Nutzung
2. **Gezielte Filter einsetzen**, um nur geschäftskritische documents zu synchronisieren
3. **Synchronisationsstatus regelmäßig überwachen** aufgrund der geplanten Ausführung
4. **Manuelle Synchronisationen auslösen**, wenn sofortige Aktualisierungen benötigt werden
5. **Kontotyp berücksichtigen**, wenn Erwartungen definiert werden

<Callout type="info">
**OneDrive-spezifische Vorteile:**
- Unterstützt sowohl persönliche als auch geschäftliche Microsoft-Konten
- Verarbeitet alle gängigen Office-Dokumentformate
- Bewahrt Dokumentstruktur und Formatierung
- Bewältigt große Dokumentbestände in Unternehmen
</Callout>

<Warning>
**Wichtige Einschränkungen:**
- Große Office-Dokumente können erheblich Zeit für die Verarbeitung benötigen
- Komplexe Excel-Formeln werden möglicherweise nicht perfekt in Markdown konvertiert
- Microsoft-API-Rate-Limits können die Synchronisierung bei großen Konten verlangsamen
</Warning>
