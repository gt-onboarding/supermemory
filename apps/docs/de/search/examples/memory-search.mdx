---
title: "Suche in Speicher‑Einträgen (/v4/search)"
description: "Suche mit minimaler Latenz, optimiert für Chatbots und konversationelle KI"
---

Die Suche in Speicher‑Einträgen (`POST /v4/search`) bietet minimale Latenz und ist für Echtzeit‑Interaktionen optimiert. Dieser Endpunkt setzt auf Geschwindigkeit statt umfangreicher Steuerungsmöglichkeiten und ist ideal für Chatbots, Q&A‑Systeme und alle Anwendungen, in denen Nutzende sofortige Antworten erwarten.

<div id="basic-search">
  ## Einfache Suche
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Supermemory from 'supermemory';

    const client = new Supermemory({
      apiKey: process.env.SUPERMEMORY_API_KEY!
    });

    const results = await client.search.memories({
      q: "machine learning applications",
      limit: 5
    });

    console.log(results)
    ```
  </Tab>

  <Tab title="Python">
    ```python
    from supermemory import Supermemory
    import os

    client = Supermemory(api_key=os.environ.get("SUPERMEMORY_API_KEY"))

    results = client.search.memories(
        q="machine learning applications",
        limit=5
    )

    console.log(results)
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    curl -X POST "https://api.supermemory.ai/v4/search" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "q": "machine learning applications",
        "limit": 5
      }'
    ```
  </Tab>
</Tabs>

<div id="chatbot-example">
  ## Chatbot-Beispiel
</div>

Optimale Konfiguration für dialogorientierte KI:

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // Für Chatbot-Antworten optimiert
    const results = await client.search.memories({
      q: userMessage,
      containerTag: userId,
      threshold: 0.6,     // Ausgewogene Relevanz
      rerank: false,      // Für höhere Geschwindigkeit überspringen
      rewriteQuery: false, // Für höhere Geschwindigkeit überspringen
      limit: 3            // Wenige, aber relevante Ergebnisse
    });

    // Schnelle Antwort im Chat
    const context = results.results
      .map(r => r.memory)
      .join('\n\n');
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # Für Chatbot-Antworten optimiert
    results = client.search.memories(
        q=user_message,
        container_tag=user_id,
        threshold=0.6,     # Ausgewogene Relevanz
        rerank=False,      # Für höhere Geschwindigkeit überspringen
        rewrite_query=False, # Für höhere Geschwindigkeit überspringen
        limit=3            # Wenige, aber relevante Ergebnisse
    )

    # Schnelle Antwort im Chat
    context = '\n\n'.join([r.memory for r in results.results])
    ```
  </Tab>
  <Tab title="cURL">

<div id="comon-use-cases">
  ## Häufige Anwendungsfälle
</div>

- **Chatbots**: Einfache Suche mit Container-Tag und niedrigem threshold
- **Q&A-Systeme**: Reranking für bessere Relevanz aktivieren
- **Knowledge Retrieval**: documents und Zusammenfassungen einbeziehen
- **Echtzeitsuche**: Rewriting und Reranking für maximale Geschwindigkeit überspringen
