---
title: "Verarbeitungsstatus"
description: "Erfahren Sie mehr über die Phasen der Inhaltsverarbeitung"
---

Nachdem Sie Inhalte hinzugefügt haben, können Sie ihren Verarbeitungsstatus prüfen:

1. `queued`: Inhalt steht zur Verarbeitung in der Warteschlange
2. `extracting`: Inhalt wird aus der Quelle extrahiert
3. `chunking`: Inhalt wird in semantische Chunks aufgeteilt
4. `embedding`: Vektor-Embeddings werden erzeugt
5. `indexing`: Wird dem Suchindex hinzugefügt
6. `done`: Verarbeitung abgeschlossen
7. `failed`: Verarbeitung fehlgeschlagen
