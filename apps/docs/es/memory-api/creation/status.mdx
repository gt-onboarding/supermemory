---
title: "Estado de procesamiento"
description: "Conoce las etapas del procesamiento de contenido"
---

Después de agregar contenido, puedes verificar su estado de procesamiento:

1. `queued`: El contenido está en cola para su procesamiento
2. `extracting`: Extrayendo el contenido de la fuente
3. `chunking`: Dividiendo el contenido en chunks semánticos
4. `embedding`: Generando embeddings vectoriales
5. `indexing`: Añadiéndolo al índice de búsqueda
6. `done`: Procesamiento completo
7. `failed`: El procesamiento falló
