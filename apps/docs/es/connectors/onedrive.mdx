Conecta OneDrive para sincronizar automáticamente documentos de Word, hojas de cálculo de Excel y presentaciones de PowerPoint en tu base de conocimientos de Supermemory. Compatible con cuentas personales y empresariales, con sincronización programada.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-create-onedrive-connection">
  ### 1. Crear la integración de OneDrive
</div>

<CodeGroup>

```typescript Typescript
import Supermemory from 'supermemory';

const client = new Supermemory({
  apiKey: process.env.SUPERMEMORY_API_KEY!
});

const connection = await client.connections.create('onedrive', {
  redirectUrl: 'https://yourapp.com/auth/onedrive/callback',
  containerTags: ['user-123', 'onedrive-sync'],
  documentLimit: 1500,
  metadata: {
    source: 'onedrive',
    accountType: 'business',
    department: 'marketing'
  }
});

// Redirigir al usuario a Microsoft OAuth
window.location.href = connection.authLink;
// Salida: Redirige al proveedor OAuth
// Salida: Redirige a https://login.microsoftonline.com/oauth2/authorize?...
```

```python Python
from supermemory import Supermemory
import os

client = Supermemory(api_key=os.environ.get("SUPERMEMORY_API_KEY"))

connection = client.connections.create(
    'onedrive',
    redirect_url='https://yourapp.com/auth/onedrive/callback',
    container_tags=['user-123', 'onedrive-sync'],
    document_limit=1500,
    metadata={
        'source': 'onedrive',
        'accountType': 'business',
        'department': 'marketing'
    }
)

# Redirigir al usuario a Microsoft OAuth
print(f'Redirect to: {connection.auth_link}')
# Salida: Redirect to: https://oauth.provider.com/...
# Salida: Redirect to: https://login.microsoftonline.com/oauth2/authorize?...
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/connections/onedrive" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "redirectUrl": "https://yourapp.com/auth/onedrive/callback",
    "containerTags": ["user-123", "onedrive-sync"],
    "documentLimit": 1500,
    "metadata": {
      "source": "onedrive",
      "accountType": "business",
      "department": "marketing"
    }
  }'

# Respuesta: {
#   "authLink": "https://login.microsoftonline.com/oauth2/authorize?...",
#   "expiresIn": "1 hour",
#   "id": "conn_od123"
# }
```

</CodeGroup>

<div id="2-handle-microsoft-oauth">
  ### 2. Gestionar OAuth de Microsoft
</div>

Después de que el usuario otorgue los permisos, Microsoft redirige a tu url de callback. La integración se establece automáticamente y comienza la sincronización inicial.

<div id="sync-mechanism">
  ## Mecanismo de sincronización
</div>

Los webhooks permiten sincronizar en tiempo real los cambios en los Documentos. También puedes iniciar una sincronización manualmente.

<div id="manual-sync-trigger">
  ### Activar sincronización manual
</div>

<CodeGroup>
  ```typescript Typescript
    // Activar sincronización inmediata para todas las integraciones de OneDrive
    await client.connections.import('onedrive');

    // Activar sincronización para un usuario específico
    await client.connections.import('onedrive', {
      containerTags: ['user-123']
    });

    console.log('Sincronización manual iniciada: los documentos se actualizarán en 5–10 minutos');
    ```
  ```python Python
    # Activar sincronización inmediata para todas las integraciones de OneDrive
    client.connections.import_('onedrive')

    # Activar sincronización para un usuario específico
    client.connections.import_(
        'onedrive',
        container_tags=['user-123']
    )

    print('Sincronización manual iniciada: los documentos se actualizarán en 5–10 minutos')
    ```
  ```bash cURL
    # Activar sincronización manual
    curl -X POST "https://api.supermemory.ai/v3/connections/onedrive/import" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"containerTags": ["user-123"]}'
    ```
</CodeGroup>

<div id="delete-connection">
  ## Eliminar integración
</div>

Elimina una integración de OneDrive cuando ya no la necesites:

<CodeGroup>

```typescript Typescript
// Eliminar por ID de la integración
const result = await client.connections.delete('connection_id_123');
console.log('Integración eliminada:', result.id);

// Eliminar por provider y container tags
const providerResult = await client.connections.deleteByProvider('onedrive', {
  containerTags: ['user-123']
});
console.log('Integración de OneDrive eliminada para el usuario');
```

```python Python
# Eliminar por ID de la integración
result = client.connections.delete('connection_id_123')
print(f'Integración eliminada: {result.id}')

# Eliminar por provider y container tags
provider_result = client.connections.delete_by_provider(
    'onedrive',
    container_tags=['user-123']
)
print('Integración de OneDrive eliminada para el usuario')
```

```bash cURL
# Eliminar por ID de la integración
curl -X DELETE "https://api.supermemory.ai/v3/connections/connection_id_123" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY"

# Eliminar por provider y container tags
curl -X DELETE "https://api.supermemory.ai/v3/connections/onedrive" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"containerTags": ["user-123"]}'
```

</CodeGroup>

<Note>
Eliminar una integración:
- Detendrá todas las sincronizaciones futuras desde OneDrive
- Revocará la autorización de OAuth
- Mantendrá los Documentos ya sincronizados en Supermemory (no se eliminarán)
</Note>

<div id="custom-microsoft-app">
  ### Aplicación personalizada de Microsoft
</div>

Para implementaciones en producción, configura tu propia aplicación de Microsoft:

<CodeGroup>
  ```typescript Typescript
    // Primero, actualiza la configuración de la organización con las credenciales de tu app de Microsoft
    await client.settings.update({
      onedriveCustomKeyEnabled: true,
      onedriveClientId: 'your-microsoft-app-id',
      onedriveClientSecret: 'your-microsoft-app-secret'
    });

    // Luego, crea integraciones usando tu app personalizada
    const connection = await client.connections.create('onedrive', {
      redirectUrl: 'https://yourapp.com/callback',
      containerTags: ['org-456', 'user-789'],
      metadata: { customApp: true }
    });
    ```
  ```python Python
    # Primero, actualiza la configuración de la organización con las credenciales de tu app de Microsoft
    client.settings.update(
        onedrive_custom_key_enabled=True,
        onedrive_client_id='your-microsoft-app-id',
        onedrive_client_secret='your-microsoft-app-secret'
    )

    # Luego, crea integraciones usando tu app personalizada
    connection = client.connections.create(
        'onedrive',
        redirect_url='https://yourapp.com/callback',
        container_tags=['org-456', 'user-789'],
        metadata={'customApp': True}
    )
    ```
  ```bash cURL
    # Actualiza la configuración de la organización
    curl -X PATCH "https://api.supermemory.ai/v3/settings" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "onedriveCustomKeyEnabled": true,
        "onedriveClientId": "your-microsoft-app-id",
        "onedriveClientSecret": "your-microsoft-app-secret"
      }'
    ```

<div id="optimization-tips">
  ### Consejos de optimización
</div>

1. **Define límites realistas de documentos** según el almacenamiento y el uso
2. **Utiliza filtros específicos** para sincronizar solo los documentos críticos para el negocio
3. **Supervisa el estado de la sincronización** con regularidad debido a su programación
4. **Lanza sincronizaciones manuales** cuando se necesiten actualizaciones inmediatas
5. **Ten en cuenta el tipo de cuenta** al establecer expectativas

<Callout type="info">
**Beneficios específicos de OneDrive:**
- Admite cuentas personales y empresariales de Microsoft
- Procesa los principales formatos de documentos de Office
- Conserva la estructura y el formato del documento
- Gestiona colecciones grandes de documentos empresariales
</Callout>

<Warning>
**Limitaciones importantes:**
- Los documentos grandes de Office pueden tardar bastante en procesarse
- Las fórmulas complejas de Excel pueden no convertirse perfectamente a Markdown
- Los límites de velocidad de la API de Microsoft pueden ralentizar la sincronización en cuentas grandes
</Warning>
