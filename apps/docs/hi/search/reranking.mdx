रीरैंकिंग खोज परिणामों की प्रासंगिकता के क्रम को बेहतर बनाने के लिए एक द्वितीयक रैंकिंग एल्गोरिदम लागू करती है। प्रारंभिक खोज के परिणाम आने के बाद, रीरैंकर आपकी क्वेरी और प्रत्येक परिणाम के बीच संबंध का विश्लेषण करता है ताकि बेहतर क्रम प्रस्तुत किया जा सके।

<div id="how-reranking-works">
  ## रीरैंकिंग कैसे काम करती है
</div>

Supermemory की रीरैंकिंग प्रक्रिया:

1. **प्रारंभिक खोज** मानक सेमांटिक समानता के आधार पर परिणाम लौटाती है
2. **रीरैंकर मॉडल** क्वेरी–परिणाम युग्मों का विश्लेषण करता है
3. **स्कोर** गहन सेमांटिक समझ के आधार पर दोबारा गणना किए जाते हैं
4. **परिणाम** नए प्रासंगिकता स्कोर के अनुसार पुन: क्रमित किए जाते हैं
5. **अंतिम परिणाम** वही संरचना बनाए रखते हैं, बस क्रम बेहतर होता है

रीरैंकर खासतौर पर इनमें प्रभावी है:

- **संदर्भ को समझना** और सूक्ष्म संबंधों को पकड़ना
- **अस्पष्ट क्वेरियों को संभालना** जिनके कई संभावित अर्थ हों
- **जटिल तकनीकी विषयों के लिए शुद्धता/प्रेसिजन में सुधार करना**
- **बेहतर रैंकिंग** जब परिणामों के प्रारंभिक स्कोर समान हों

<div id="basic-reranking-comparison">
  ## बेसिक रीरैंकिंग तुलना
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Supermemory from 'supermemory';

    const client = new Supermemory({
      apiKey: process.env.SUPERMEMORY_API_KEY!
    });

    // रीरैंकिंग के बिना खोज
    const standardResults = await client.search.documents({
      q: "neural network optimization techniques",
      rerank: false,
      limit: 5
    });

    // रीरैंकिंग के साथ खोज
    const rerankedResults = await client.search.documents({
      q: "neural network optimization techniques",
      rerank: true,
      limit: 5
    });

    console.log("मानक शीर्ष परिणाम:", standardResults.results[0].score);
    console.log("रीरैंक किया गया शीर्ष परिणाम:", rerankedResults.results[0].score);
    ```
  </Tab>

  <Tab title="Python">
    ```python
    from supermemory import Supermemory
    import os

    client = Supermemory(api_key=os.environ.get("SUPERMEMORY_API_KEY"))

    # रीरैंकिंग के बिना खोज
    standard_results = client.search.documents(
        q="neural network optimization techniques",
        rerank=False,
        limit=5
    )

    # रीरैंकिंग के साथ खोज
    reranked_results = client.search.documents(
        q="neural network optimization techniques",
        rerank=True,
        limit=5
    )

    print("मानक शीर्ष परिणाम:", standard_results.results[0].score)
    print("रीरैंक किया गया शीर्ष परिणाम:", reranked_results.results[0].score)
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    # रीरैंकिंग के बिना
    echo "मानक रैंकिंग:"

      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "q": "neural network optimization techniques",
        "rerank": false,
        "limit": 3
      }' | jq '.results[0] | {title, score}'

      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "q": "neural network optimization techniques",
        "rerank": true,
        "limit": 3
      }' | jq '.results[0] | {title, score}'
    ```

<div id="domain-specific-reranking">
  ## डोमेन-विशिष्ट रीरैंकिंग
</div>

रीरैंकिंग डोमेन-विशिष्ट संबंधों को समझती है:

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // चिकित्सा डोमेन क्वेरी
    const medicalResults = await client.search.documents({
      q: "diabetes treatment insulin resistance metformin",
      rerank: true,
      filters: {
        AND: [
          { key: "domain", value: "medical", negate: false }
        ]
      },
      limit: 10
    });

    // रीरैंकर चिकित्सा संबंधों को समझता है:
    // - डायबिटीज के प्रकार और उपचार
    // - इंसुलिन रेसिस्टेंस के तंत्र
    // - डायबिटीज प्रबंधन में मेटफॉर्मिन की भूमिका
    ```
  </Tab>
  <Tab title="Python">
    ```python

    medical_results = client.search.documents(

            "AND": [
                {"key": "domain", "value": "medical", "negate": False}
    ```
