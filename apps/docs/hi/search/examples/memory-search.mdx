Memories search (`POST /v4/search`) वास्तविक‑समय इंटरैक्शन के लिए अनुकूलित, न्यूनतम लेटेंसी वाली खोज प्रदान करता है। यह endpoint विस्तृत नियंत्रण की तुलना में गति को प्राथमिकता देता है, इसलिए यह चैटबॉट्स, Q&A सिस्टम, और ऐसे किसी भी एप्लिकेशन के लिए उपयुक्त है जहाँ उपयोगकर्ता त्वरित प्रतिक्रिया की अपेक्षा करते हैं।

<div id="basic-search">
  ## मूलभूत खोज
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Supermemory from 'supermemory';

    const client = new Supermemory({
      apiKey: process.env.SUPERMEMORY_API_KEY!
    });

    const results = await client.search.memories({
      q: "machine learning applications",
      limit: 5
    });

    console.log(results)
    ```
  </Tab>

  <Tab title="Python">
    ```python
    from supermemory import Supermemory
    import os

    client = Supermemory(api_key=os.environ.get("SUPERMEMORY_API_KEY"))

    results = client.search.memories(
        q="machine learning applications",
        limit=5
    )

    print(results)
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    curl -X POST "https://api.supermemory.ai/v4/search" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "q": "machine learning applications",
        "limit": 5
      }'
    ```

<div id="chatbot-example">
  ## चैटबॉट उदाहरण
</div>

संवादी AI के लिए सर्वोत्तम कॉन्फ़िगरेशन:

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // चैटबॉट प्रतिक्रियाओं के लिए ऑप्टिमाइज़्ड
    const results = await client.search.memories({
      q: userMessage,
      containerTag: userId,
      threshold: 0.6,     // संतुलित प्रासंगिकता
      rerank: false,      // गति के लिए छोड़ें
      rewriteQuery: false, // गति के लिए छोड़ें
      limit: 3            // कम, पर प्रासंगिक परिणाम
    });

    // चैट के लिए त्वरित प्रतिक्रिया
    const context = results.results
      .map(r => r.memory)
      .join('\n\n');
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # चैटबॉट प्रतिक्रियाओं के लिए ऑप्टिमाइज़्ड
    results = client.search.memories(
        q=user_message,
        container_tag=user_id,
        threshold=0.6,     # संतुलित प्रासंगिकता
        rerank=False,      # गति के लिए छोड़ें
        rewrite_query=False, # गति के लिए छोड़ें
        limit=3            # कम, पर प्रासंगिक परिणाम
    )

    # चैट के लिए त्वरित प्रतिक्रिया
    context = '\n\n'.join([r.memory for r in results.results])
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # चैटबॉट प्रतिक्रियाओं के लिए ऑप्टिमाइज़्ड
    curl -X POST "https://api.supermemory.ai/v4/search" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "q": "user question here",
        "containerTag": "user_123",
        "threshold": 0.6,
        "rerank": false,
        "rewriteQuery": false,
        "limit": 3
      }'
    ```
  </Tab>

<div id="comon-use-cases">
  ## सामान्य उपयोग के मामले
</div>

- **Chatbots**: कंटेनर टैग के साथ बेसिक सर्च और कम threshold
- **Q&A Systems**: बेहतर प्रासंगिकता के लिए रीरैंकिंग जोड़ें
- **Knowledge Retrieval**: documents और summaries शामिल करें
- **Real-time Search**: अधिकतम गति के लिए rewriting और रीरैंकिंग को छोड़ें
