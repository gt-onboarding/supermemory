<div id="usage">
  ## उपयोग
</div>

```python
import os
from supermemory import Supermemory

client = supermemory(
    api_key=os.environ.get("SUPERMEMORY_API_KEY"),  # यह डिफ़ॉल्ट है और इसे छोड़ा जा सकता है
)

response = client.search.documents(
    q="python से संबंधित documents",
)
print(response.results)
```

<div id="usage">
  ## उपयोग
</div>

```js
import supermemory from 'supermemory';

const client = new supermemory({
  apiKey: process.env['SUPERMEMORY_API_KEY'], // यह डिफ़ॉल्ट है और इसे छोड़ा जा सकता है
});

async function main() {
  const response = await client.search.documents({ q: 'python से संबंधित documents' });

  console.debug(response.results);
}

main();
```
