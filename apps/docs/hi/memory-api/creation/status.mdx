---
title: "प्रोसेसिंग स्टेटस"
description: "कंटेंट प्रोसेसिंग के चरणों के बारे में जानें"
---

कंटेंट जोड़ने के बाद, आप उसका processing status देख सकते हैं:

1. `queued`: कंटेंट प्रोसेसिंग के लिए कतारबद्ध है
2. `extracting`: स्रोत से कंटेंट निकाला जा रहा है
3. `chunking`: कंटेंट को अर्थपूर्ण चंक में बाँटना
4. `embedding`: वेक्टर एम्बेडिंग जनरेट करना
5. `indexing`: सर्च इंडेक्स में जोड़ना
6. `done`: प्रोसेसिंग पूर्ण
7. `failed`: प्रोसेसिंग विफल
