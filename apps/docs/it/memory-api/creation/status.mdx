---
title: "Stato di elaborazione"
description: "Scopri le fasi dell'elaborazione dei contenuti"
---

Dopo aver aggiunto contenuti, puoi verificarne lo stato di elaborazione:

1. `queued`: Il contenuto è in coda per l'elaborazione
2. `extracting`: Estrazione del contenuto dalla sorgente
3. `chunking`: Suddivisione del contenuto in chunk semantici
4. `embedding`: Generazione di embedding vettoriali
5. `indexing`: Aggiunta all’indice di ricerca
6. `done`: Elaborazione completata
7. `failed`: Elaborazione non riuscita
