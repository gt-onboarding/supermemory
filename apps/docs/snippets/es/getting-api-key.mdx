Dirígete a la [Plataforma para desarrolladores de Supermemory](https://console.supermemory.ai), creada para ayudarte a supervisar y gestionar todos los aspectos de la API.

<Accordion title="Obtener una clave de API" icon="sparkles">
  <Steps>
    <Step title="Crear una cuenta" icon="circle-user">
      Se creará una cuenta automáticamente en tu primer inicio de sesión.

      ![Dev Platform Login Pn](/images/dev-platform-login.png)
    </Step>
    <Step title="Crear una clave de API" icon="key">
      <Steps>
        <Step title="Ir a API Keys">
          ![Dev Platform Api Keys Pn](/images/dev-platform-api-keys.png)

          Haz clic en `Create API Key`
        </Step>
        <Step title="Elegir nombre y vencimiento (opcional)">
          ![Dev Platform Create Key Pn](/images/dev-platform-create-key.png)

          Haz clic en `Create`
        </Step>
        <Step title="Copiar la nueva clave">
          ![Dev Platform Copy Key Pn](/images/dev-platform-copy-key.png)

          Haz clic en <Icon icon="clipboard" /> para copiar
        </Step>
      </Steps>
    </Step>
  </Steps>
</Accordion>
