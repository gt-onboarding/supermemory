[supermemory の Developer Platform](https://console.supermemory.ai) にアクセスしてください。API のあらゆる側面を監視・管理できるよう設計されています。

<Accordion title="APIキーの取得" icon="sparkles">
  <Steps>
    <Step title="アカウントを作成" icon="circle-user">
      初回ログイン時に自動でアカウントが作成されます。

      ![Dev Platform Login Pn](/images/dev-platform-login.png)
    </Step>
    <Step title="APIキーを作成" icon="key">
      <Steps>
        <Step title="API Keys に移動">
          ![Dev Platform Api Keys Pn](/images/dev-platform-api-keys.png)

          「`Create API Key`」をクリック
        </Step>
        <Step title="名前と有効期限を設定（任意）">
          ![Dev Platform Create Key Pn](/images/dev-platform-create-key.png)

          「`Create`」をクリック
        </Step>
        <Step title="新しいキーをコピー">
          ![Dev Platform Copy Key Pn](/images/dev-platform-copy-key.png)

          コピーするには <Icon icon="clipboard" /> をクリック
        </Step>
      </Steps>
    </Step>
  </Steps>
</Accordion>

完了！これで APIキーを使って supermemory API へのリクエストを認証できます。
