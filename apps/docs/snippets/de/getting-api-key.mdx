Gehe zur [Developer Platform von Supermemory](https://console.supermemory.ai), die dafür entwickelt wurde, jeden Aspekt der API zu überwachen und zu verwalten.

<Accordion title="Einen API-Schlüssel erhalten" icon="sparkles">
  <Steps>
    <Step title="Konto erstellen" icon="circle-user">
      Beim ersten Anmelden wird automatisch ein Konto erstellt.

      ![Dev Platform Login Pn](/images/dev-platform-login.png)
    </Step>
    <Step title="API-Schlüssel erstellen" icon="key">
      <Steps>
        <Step title="Zu API Keys navigieren">
          ![Dev Platform Api Keys Pn](/images/dev-platform-api-keys.png)

          Klicke auf `Create API Key`
        </Step>
        <Step title="Name und Ablaufdatum wählen (optional)">
          ![Dev Platform Create Key Pn](/images/dev-platform-create-key.png)

          Klicke auf `Create`
        </Step>
        <Step title="Neuen Schlüssel kopieren">
          ![Dev Platform Copy Key Pn](/images/dev-platform-copy-key.png)

          Klicke auf <Icon icon="clipboard" /> zum Kopieren
        </Step>
      </Steps>
    </Step>
  </Steps>
</Accordion>

Fertig! Du kannst jetzt deinen API-Schlüssel verwenden, um Anfragen an die supermemory API zu authentifizieren.
