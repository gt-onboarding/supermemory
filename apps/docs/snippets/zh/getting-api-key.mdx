前往 [supermemory 的 Developer Platform](https://console.supermemory.ai)，该平台可帮助你监控并管理 API 的各个方面。

<Accordion title="获取 API 密钥" icon="sparkles">
  <Steps>
    <Step title="创建账户" icon="circle-user">
      你在首次登录时会自动创建账户。

      ![Dev Platform Login Pn](/images/dev-platform-login.png)
    </Step>
    <Step title="创建 API 密钥" icon="key">
      <Steps>
        <Step title="前往 API Keys">
          ![Dev Platform Api Keys Pn](/images/dev-platform-api-keys.png)

          点击 `Create API Key`
        </Step>
        <Step title="选择名称和过期时间（可选）">
          ![Dev Platform Create Key Pn](/images/dev-platform-create-key.png)

          点击 `Create`
        </Step>
        <Step title="复制新密钥">
          ![Dev Platform Copy Key Pn](/images/dev-platform-copy-key.png)

          点击 <Icon icon="clipboard" /> 进行复制
        </Step>
      </Steps>
    </Step>
  </Steps>
