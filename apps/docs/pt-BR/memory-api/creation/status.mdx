---
title: "Status de processamento"
description: "Conheça as etapas do processamento de conteúdo"
---

Depois de adicionar conteúdo, você pode verificar o status de processamento:

1. `queued`: Conteúdo em fila para processamento
2. `extracting`: Extraindo conteúdo da fonte
3. `chunking`: Segmentando o conteúdo em chunks semânticos
4. `embedding`: Gerando embeddings vetoriais
5. `indexing`: Adicionando ao índice de pesquisa
6. `done`: Processamento concluído
7. `failed`: Processamento com falha
