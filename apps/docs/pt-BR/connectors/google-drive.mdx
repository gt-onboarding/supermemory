Conecte o Google Drive para sincronizar documents na sua base de conhecimento do Supermemory com autenticação OAuth e suporte a aplicativos personalizados.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-create-google-drive-connection">
  ### 1. Criar integração com o Google Drive
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Supermemory from 'supermemory';

    const client = new Supermemory({
      apiKey: process.env.SUPERMEMORY_API_KEY!
    });

    const connection = await client.connections.create('google-drive', {
      redirectUrl: 'https://yourapp.com/auth/google-drive/callback',
      containerTags: ['user-123', 'gdrive-sync'],
      documentLimit: 3000,
      metadata: {
        source: 'google-drive',
        department: 'engineering'
      }
    });

    // Redirecione o usuário para o Google OAuth
    window.location.href = connection.authLink;
    console.log('A autorização expira em:', connection.expiresIn);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from supermemory import Supermemory
    import os

    client = Supermemory(api_key=os.environ.get("SUPERMEMORY_API_KEY"))

    connection = client.connections.create(
        'google-drive',
        redirect_url='https://yourapp.com/auth/google-drive/callback',
        container_tags=['user-123', 'gdrive-sync'],
        document_limit=3000,
        metadata={
            'source': 'google-drive',
            'department': 'engineering'
        }
    )

    # Redirecione o usuário para o Google OAuth
    print(f'Redirecionar para: {connection.auth_link}')
    print(f'Expira em: {connection.expires_in}')
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    curl -X POST "https://api.supermemory.ai/v3/connections/google-drive" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "redirectUrl": "https://yourapp.com/auth/google-drive/callback",
        "containerTags": ["user-123", "gdrive-sync"],
        "documentLimit": 3000,
        "metadata": {
          "source": "google-drive",
          "department": "engineering"
        }
      }'
    ```
  </Tab>
</Tabs>

<div id="2-handle-oauth-callback">
  ### 2. Tratar o callback do OAuth
</div>

Após o usuário conceder as permissões, o Google redireciona para sua url de callback. A integração é estabelecida automaticamente.

<div id="3-check-connection-status">
  ### 3. Verificar o status da integração
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // Obter detalhes da integração
    const connection = await client.connections.getByTags('google-drive', {
      containerTags: ['user-123', 'gdrive-sync']
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # Obter detalhes da integração
    connection = client.connections.get_by_tags(
        'google-drive',
        container_tags=['user-123', 'gdrive-sync']
    )

    # Listar documents sincronizados
    documents = client.connections.list_documents(
        'google-drive',
        container_tags=['user-123', 'gdrive-sync']
    )
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # Obter integrações por provider e tags
    curl -X POST "https://api.supermemory.ai/v3/connections/list" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "containerTags": ["user-123", "gdrive-sync"],
        "provider": "google-drive"
      }'

    # Listar documents sincronizados
    curl -X POST "https://api.supermemory.ai/v3/documents/list" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "containerTags": ["user-123", "gdrive-sync"],
        "source": "google-drive"
      }'
    ```
  </Tab>
</Tabs>

<div id="supported-document-types">
  ## Tipos de documentos compatíveis
</div>

Com base nas definições de tipos da API, os documents do Google Drive são identificados pelos seguintes tipos:

- `google_doc` - Google Docs
- `google_slide` - Google Slides
- `google_sheet` - Google Sheets

<div id="delete-connection">
  ### Excluir integração
</div>

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // Excluir por ID da integração
    const result = await client.connections.deleteByID('connection_id_123');
    console.log('Integração excluída:', result.id);

    // Excluir por provider e container tags
    const providerResult = await client.connections.deleteByProvider('google-drive', {
      containerTags: ['user-123']
    });
    console.log('Integração do provider excluída:', providerResult.id);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # Excluir por ID da integração
    result = client.connections.delete_by_id('connection_id_123')
    print(f'Integração excluída: {result.id}')

    # Excluir por provider e container tags
    provider_result = client.connections.delete_by_provider(
        'google-drive',
        container_tags=['user-123']
    )
    print(f'Integração do provider excluída: {provider_result.id}')
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # Excluir por ID da integração
    curl -X DELETE "https://api.supermemory.ai/v3/connections/connection_id_123" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY"

    # Resposta: {"id": "connection_id_123", "provider": "google-drive"}

    # Excluir por provider e container tags
    curl -X DELETE "https://api.supermemory.ai/v3/connections/google-drive" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "containerTags": ["user-123"]
      }'

    # Resposta: {"id": "conn_gd123", "provider": "google-drive"}
    ```
  </Tab>
</Tabs>

<Note>
Excluir uma integração:
- Interrompe todas as sincronizações futuras do Google Drive
- Remove a autorização OAuth
- Mantém os documents já sincronizados na Supermemory (eles não serão excluídos)
</Note>

<div id="custom-oauth-application">
  ### Aplicativo OAuth personalizado
</div>

Configure seu próprio app Google OAuth usando a API de configurações:

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // Atualize as configurações da organização com seu app Google OAuth
    await client.settings.update({
      googleDriveCustomKeyEnabled: true,
      googleDriveClientId: 'your-google-client-id.googleusercontent.com',
      googleDriveClientSecret: 'your-google-client-secret'
    });

    // Obtenha as configurações atuais
    const settings = await client.settings.get();
    console.log('Chave personalizada do Google Drive ativada:', settings.googleDriveCustomKeyEnabled);
    console.log('Client ID configurado:', !!settings.googleDriveClientId);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # Atualize as configurações da organização com seu app Google OAuth
    client.settings.update(
        google_drive_custom_key_enabled=True,
        google_drive_client_id='your-google-client-id.googleusercontent.com',
        google_drive_client_secret='your-google-client-secret'
    )

    # Obtenha as configurações atuais
    settings = client.settings.get()
    print(f'Chave personalizada do Google Drive ativada: {settings.google_drive_custom_key_enabled}')
    print(f'Client ID configurado: {bool(settings.google_drive_client_id)}')
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # Atualize as configurações da organização
    curl -X PATCH "https://api.supermemory.ai/v3/settings" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "googleDriveCustomKeyEnabled": true,
        "googleDriveClientId": "your-google-client-id.googleusercontent.com",
        "googleDriveClientSecret": "your-google-client-secret"
      }'

    # Obtenha as configurações atuais
    curl -X GET "https://api.supermemory.ai/v3/settings" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY"
    ```
  </Tab>
</Tabs>

<div id="document-filtering">
  ### Filtragem de documentos
</div>

Configure a filtragem usando a API de configurações:

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    await client.settings.update({
      shouldLLMFilter: true,
      filterPrompt: "Only sync important business documents",
      includeItems: {
        // Seus padrões de inclusão
      },
      excludeItems: {
        // Seus padrões de exclusão
      }
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    client.settings.update(
        should_llm_filter=True,
        filter_prompt="Only sync important business documents",
        include_items={
            # Seus padrões de inclusão
        },
        exclude_items={
            # Seus padrões de exclusão
        }
    )
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # Configurar a filtragem de documentos
    curl -X PATCH "https://api.supermemory.ai/v3/settings" \
      -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "shouldLLMFilter": true,
        "filterPrompt": "Only sync important business documents",
        "includeItems": {
          "patterns": ["*.pdf", "*.docx"],
          "folders": ["Important Documents", "Projects"]
        },
        "excludeItems": {
          "patterns": ["*.tmp", "*.backup"],
          "folders": ["Archive", "Trash"]
        }
      }'

    # Resposta: {
    #   "shouldLLMFilter": true,
    #   "filterPrompt": "Only sync important business documents",
    #   "includeItems": {...},
    #   "excludeItems": {...}
    # }
    ```
  </Tab>
</Tabs>

<Warning>
**Notas importantes:**
- Tokens OAuth podem expirar — verifique o campo `expiresAt`
- O processamento de documentos ocorre de forma assíncrona
- Use tags de contêiner de forma consistente para a filtragem
- Monitore o status dos documentos para identificar sincronizações com falha
</Warning>
