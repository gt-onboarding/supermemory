Aprenda a adicionar conteúdo de texto básico ao Supermemory com exemplos simples e práticos.

<div id="add-simple-text">
  ## Adicionar texto simples
</div>

A operação mais básica — adicionar conteúdo de texto puro.

<CodeGroup>

```typescript TypeScript
const response = await client.memories.add({
  content: "Artificial intelligence is transforming how we work and live"
});

console.log(response);
// Output: { id: "abc123", status: "queued" }
```

```python Python
response = client.memories.add(
    content="Artificial intelligence is transforming how we work and live"
)

print(response)
# Output: {"id": "abc123", "status": "queued"}
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Artificial intelligence is transforming how we work and live"
  }'
```

</CodeGroup>

<div id="add-with-container-tags">
  ## Adicionar com tags de contêiner
</div>

Agrupe conteúdos relacionados usando tags de contêiner.

<CodeGroup>

```typescript TypeScript
const response = await client.memories.add({
  content: "Q4 2024 revenue exceeded projections by 15%",
  containerTag: "financial_reports"
});

console.log(response.id);
// Saída: xyz789
```

```python Python
response = client.memories.add(
    content="Q4 2024 revenue exceeded projections by 15%",
    container_tag="financial_reports"
)

print(response['id'])
# Saída: xyz789
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Q4 2024 revenue exceeded projections by 15%",
    "containerTag": "financial_reports"
  }'

# Resposta: {"id": "xyz789", "status": "queued"}
```

</CodeGroup>

<div id="add-with-metadata">
  ## Adicionar com metadata
</div>

Anexe metadata para melhorar a busca e a filtragem.

<CodeGroup>

```typescript TypeScript
await client.memories.add({
  content: "New onboarding flow reduces drop-off by 30%",
  containerTag: "product_updates",
  metadata: {
    impact: "high",
    team: "product"
  }
});
```

```python Python
client.memories.add(
    content="New onboarding flow reduces drop-off by 30%",
    container_tag="product_updates",
    metadata={
        "impact": "high",
        "team": "product"
    }
)
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "New onboarding flow reduces drop-off by 30%",
    "containerTag": "product_updates",
    "metadata": {"impact": "high", "team": "product"}
  }'
```

</CodeGroup>

<div id="add-multiple-documents">
  ## Adicionar vários documents
</div>

Processe vários documents relacionados.

<CodeGroup>

```typescript TypeScript
const notes = [
  "Discussão sobre redesign da API",
  "Auditoria de segurança no próximo mês",
  "Nova contratação começando na segunda"
];

const results = await Promise.all(
  notes.map(note =>
    client.memories.add({
      content: note,
      containerTag: "meeting_2024_01_15"
    })
  )
);
```

```python Python
notes = [
    "Discussão sobre redesign da API",
    "Auditoria de segurança no próximo mês",
    "Nova contratação começando na segunda"
]

for note in notes:
    client.memories.add(
        content=note,
        container_tag="meeting_2024_01_15"
    )
```

```bash cURL
# Adicione cada nota com requisições separadas
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "Discussão sobre redesign da API", "containerTag": "meeting_2024_01_15"}'
```

</CodeGroup>

<div id="add-urls">
  ## Adicionar URLs
</div>

Procese páginas da web, vídeos do YouTube e outras URLs automaticamente.

<CodeGroup>

```typescript TypeScript
// Página da web
await client.memories.add({
  content: "https://example.com/article",
  containerTag: "articles"
});

// Vídeo do YouTube (transcrito automaticamente)
await client.memories.add({
  content: "https://youtube.com/watch?v=dQw4w9WgXcQ",
  containerTag: "videos"
});

// Google Docs
await client.memories.add({
  content: "https://docs.google.com/document/d/abc123/edit",
  containerTag: "docs"
});
```

```python Python
# Página da web
client.memories.add(
    content="https://example.com/article",
    container_tag="articles"
)

# Vídeo do YouTube (transcrito automaticamente)
client.memories.add(
    content="https://youtube.com/watch?v=dQw4w9WgXcQ",
    container_tag="videos"
)

# Google Docs
client.memories.add(
    content="https://docs.google.com/document/d/abc123/edit",
    container_tag="docs"
)
```

```bash cURL
# Página da web
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "https://example.com/article", "containerTag": "articles"}'

# Vídeo do YouTube
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "https://youtube.com/watch?v=dQw4w9WgXcQ", "containerTag": "videos"}'
```

</CodeGroup>

<div id="add-markdown-content">
  ## Adicionar conteúdo Markdown
</div>

O Supermemory preserva a formatação de Markdown.

<CodeGroup>

```typescript TypeScript
const markdown = `
# Documentação do Projeto

## Recursos
- **Sincronização em tempo real**
- **Busca com IA**
- **Segurança corporativa**
`;

await client.memories.add({
  content: markdown,
  containerTag: "docs"
});
```

```python Python
markdown = """
# Documentação do Projeto

## Recursos
- **Sincronização em tempo real**
- **Busca com IA**
- **Segurança corporativa**
"""

client.memories.add(
    content=markdown,
    container_tag="docs"
)
```

```bash cURL
curl -X POST "https://api.supermemory.ai/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "# Documentação do Projeto\n\n## Recursos\n- **Sincronização em tempo real**\n- **Busca com IA**", "containerTag": "docs"}'
```

</CodeGroup>
