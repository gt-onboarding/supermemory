<div id="whats-new-this-month">
  ## Novidades deste mês
</div>

**Setembro de 2025** foi um grande mês para a Supermemory:

<div id="for-developers">
  ### Para desenvolvedores
</div>

* **Reescrita completa da documentação** - Documentação totalmente nova com receitas práticas e exemplos de produção
* **Integração nativa do AI SDK** - Use o Supermemory diretamente com o Vercel AI SDK
* **APIs aprimoradas** - Novas operações em lote e endpoints de pesquisa otimizados

<div id="for-users">
  ### Para usuários
</div>

* **Revisão da Visual Console** - Novo layout em grade (masonry) e Cards de pré-visualização redesenhados
* **Navegação por memory aprimorada** - Visualização em grafo interativa com desempenho via WebGL
* **Experiência móvel aprimorada** - Design responsivo melhor em todos os recursos
