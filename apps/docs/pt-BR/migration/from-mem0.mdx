Migrar do Mem0.ai para o Supermemory é simples. Este guia explica como exportar suas memórias do Mem0 e importá-las para o Supermemory.

<div id="why-migrate-to-supermemory">
  ## Por que migrar para o Supermemory?
</div>

O Supermemory oferece recursos avançados em relação ao Mem0.ai:

* **Memory Router** para integração com LLM sem escrever código
* Arquitetura de **grafo de conhecimento** para relacionamentos de contexto mais robustos
* **Vários tipos de conteúdo** (URLs, PDFs, imagens, vídeos)
* **Plano gratuito generoso** (100k tokens) com preços acessíveis
* **Diversas opções de integração** (API, Router, MCP, SDKs)

<div id="quick-migration-all-in-one">
  ## Migração rápida (tudo‑em‑um)
</div>

Conclua a migração em um único script:

```python
from mem0 import MemoryClient
from supermemory import Supermemory
import json, time

# Exportar do Mem0
mem0 = MemoryClient(api_key="sua_chave_api_mem0")
export = mem0.create_memory_export(
    schema={"type": "object", "properties": {"memories": {"type": "array", "items": {"type": "object"}}}},
    filters={}
)
time.sleep(5)
data = mem0.get_memory_export(memory_export_id=export["id"])

# Importar para o Supermemory
supermemory = Supermemory(api_key="sua_chave_api_supermemory")
for memory in data["memories"]:
    if memory.get("content"):
        supermemory.memories.add(
            content=memory["content"],
            container_tags=["imported_from_mem0"]
        )
        print(f"✅ {memory['content'][:50]}...")

print("Migração concluída!")
```

<div id="step-by-step-migration">
  ## Migração passo a passo
</div>

<Steps>
  <Step title="Exportar do Mem0.ai">
    O Mem0 oferece duas maneiras de exportar suas memórias:

    ### Opção 1: Exportar pelo Dashboard (recomendado)

    1. Acesse seu [dashboard do Mem0](https://app.mem0.ai)
    2. Vá até a seção de exportação
    3. Baixe suas memórias em JSON

    ### Opção 2: Exportar via API

    Script simples para exportar todas as suas memórias do Mem0:

    ```python
    from mem0 import MemoryClient
    import json
    import time

    # Conectar ao Mem0
    client = MemoryClient(api_key="your_mem0_api_key")

    # Criar tarefa de exportação
    schema = {
        "type": "object",
        "properties": {
            "memories": {
                "type": "array",
                "items": {
                    "type": "object",
                    "properties": {
                        "id": {"type": "string"},
                        "content": {"type": "string"},
                        "metadata": {"type": "object"},
                        "created_at": {"type": "string"}
                    }
                }
            }
        }
    }

    response = client.create_memory_export(schema=schema, filters={})
    export_id = response["id"]

    # Aguardar e recuperar
    print("Exportando memórias...")
    time.sleep(5)
    export_data = client.get_memory_export(memory_export_id=export_id)

    # Salvar em arquivo
    with open("mem0_export.json", "w") as f:
        json.dump(export_data, f, indent=2)

    print(f"Exportadas {len(export_data['memories'])} memórias")
    ```
  </Step>

  <Step title="Configurar o Supermemory">
    Crie sua conta no Supermemory e obtenha sua chave da API:

    1. Cadastre-se em [console.supermemory.ai](https://console.supermemory.ai)
    2. Crie um novo projeto
    3. Gere uma chave da API pelo dashboard

    ```bash
    # Defina sua variável de ambiente
    export SUPERMEMORY_API_KEY="your_supermemory_api_key"
    ```
  </Step>

  <Step title="Importar para o Supermemory">
    Script simples para importar suas memórias do Mem0 para o Supermemory:

    ```python
    import json
    from supermemory import Supermemory

    # Carregar sua exportação do Mem0
    with open("mem0_export.json", "r") as f:
        mem0_data = json.load(f)

    # Conectar ao Supermemory
    client = Supermemory(api_key="your_supermemory_api_key")

    # Importar memórias
    for memory in mem0_data["memories"]:
        content = memory.get("content", "")

        # Ignorar memórias vazias
        if not content:
            continue

        # Importar para o Supermemory
        try:
            result = client.memories.add(
                content=content,
                container_tags=["imported_from_mem0"],
                metadata={
                    "source": "mem0",
                    "created_at": memory.get("created_at"),
                    **(memory.get("metadata") or {})
                }
            )
            print(f"Importado: {content[:50]}...")
        except Exception as e:
            print(f"Falha: {e}")

    print("Migração concluída!")
    ```
  </Step>
</Steps>

<div id="using-memory-router-easiest-migration">
  ## Usando o Memory Router (Migração mais fácil)
</div>

Para o caminho de migração mais simples, use o Memory Router da Supermemory, que exige mudanças mínimas no código:

<CodeGroup>
  ```python Before (Mem0 + OpenAI)
  from openai import OpenAI
  from mem0 import MemoryClient

  # Dois clientes separados são necessários
  openai = OpenAI(api_key="sk-...")
  memory = MemoryClient(api_key="mem0_key")

  # Gerenciamento de memory manual
  context = memory.search("user preferences", user_id="alice")
  messages = [
      {"role": "system", "content": f"Context: {context}"},
      {"role": "user", "content": "What are my preferences?"}
  ]

  response = openai.chat.completions.create(
      model="gpt-5",
      messages=messages
  )
  ```

  ```python After (Supermemory Router)
  from openai import OpenAI

  # Um único cliente com gerenciamento automático de memory
  client = OpenAI(
      api_key="sk-...",
      base_url="https://api.supermemory.ai/v3/https://api.openai.com/v1",
      default_headers={
          "x-supermemory-api-key": "your_supermemory_key",
          "x-supermemory-user-id": "alice"
      }
  )

  # Memórias gerenciadas automaticamente!
  response = client.chat.completions.create(
      model="gpt-5",
      messages=[{"role": "user", "content": "What are my preferences?"}]
  )
  ```
</CodeGroup>
